📚 Sobre o Projeto

Conceitos de POO Implementados:

  • ✅ Encapsulamento: Atributos privados com getters/setters
  • ✅ Herança: Classes derivadas de Entidade e Edificio
  • ✅ Polimorfismo: Métodos sobrescritos nas subclasses
  • ✅ Abstração: Classes abstratas (Entidade, Edificio)
  • ✅ Composição: Colônia contém Colonos, Edifícios e Recursos
  • ✅ Agregação: Relacionamentos entre objetos

Arquitetura:

  • 🏗️ Padrão MVC: Model-View-Controller
  • 💾 Persistência: Pickle para salvar/carregar
  • 🌐 Web Framework: Bottle (Python)
